💡 Hướng dẫn: Cài đặt Bridge cục bộ để khởi chạy lệnh AI.

Chạy 2 lệnh này trên Terminal máy Linux của bạn:

  1. Tải mã nguồn và vào thư mục: 
    git clone https://github.com/nghuyhoang29801/router_cli_ai.git && cd router_cli_ai
  2. Chạy lệnh cài đặt tự động: 
    chmod +x install.sh && ./install.sh
  3. Sau đó, tìm và mở Bridge App từ menu ứng dụng của hệ điều hành.
