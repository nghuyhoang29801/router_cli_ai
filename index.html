💡 Hướng dẫn: Cài đặt Bridge cục bộ để khởi chạy lệnh AI.

Chạy lệnh này trên Terminal máy Linux:

  1. Tải mã nguồn: 
    git clone https://github.com/nghuyhoang29801/router_cli_ai.git && cd router_cli_ai
  2. Cài đặt tự động: 
    chmod +x install.sh && ./install.sh
  3. Mở Bridge App từ menu ứng dụng.

Chạy lệnh này trên PowerShell (Admin):

  1. Tải mã nguồn: 
    git clone https://github.com/nghuyhoang29801/router_cli_ai.git; cd "Bridge App"
  2. Cho phép chạy script và cài đặt: 
    Set-ExecutionPolicy RemoteSigned -Scope CurrentUser; .\install.ps1
  3. Mở shortcut Bridge App trên màn hình Desktop.
